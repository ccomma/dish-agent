【退款与售后规则】

一、退菜规则
1. 菜品上桌前可全额退款，无需手续费
2. 菜品上桌后30分钟内，如味道不满意可申请换菜，差价多退少补
3. 菜品上桌后超过30分钟原则上不退（质量问题除外）
4. 以下情况可随时退菜：异物、变质、严重少料、做错订单

二、退款规则
1. 审核通过后3-7个工作日原路返回
2. 上桌前退菜全额退款，上桌后按实际情况协商
3. 门店原因订单错误、超时送达、质量严重问题可优先退款

三、投诉与售后
1. 热线电话：400-888-9999
2. 在线客服：APP 内“帮助与客服”
3. 工作时间：9:00-22:00
4. 紧急投诉2小时响应，一般投诉24小时处理
